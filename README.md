# Temperature Converter – Project Overview

The Temperature Converter is a simple application developed using the C programming language that converts temperature values between Celsius and Fahrenheit. It is designed to help users quickly and accurately convert temperatures from one scale to another.

The main objective of this project is to demonstrate the practical application of basic C programming concepts such as variables, user input/output, arithmetic operations, and conditional statements. The program provides a menu-driven interface where users can choose the type of conversion they want to perform.

The system accepts a temperature value from the user, applies the appropriate conversion formula, and displays the converted temperature. This project is useful for students learning programming fundamentals and understanding how mathematical formulas can be implemented in code.

## Objectives

1. To convert temperature values between Celsius and Fahrenheit.
2. To provide accurate and fast temperature conversions.
3. To demonstrate the use of arithmetic formulas in programming.
4. To improve understanding of user input and output operations in C.
5. To develop a simple menu-driven application.

## Features

* Celsius to Fahrenheit Conversion
* Fahrenheit to Celsius Conversion
* User-Friendly Menu Interface
* Accurate Temperature Calculation
* Simple and Easy-to-Understand Code

## Technologies Used

* Programming Language: C
* Compiler: GCC / Turbo C / CodeBlocks
* Concepts Used: Variables, Operators, Input/Output Functions, Switch-Case Statements

## Expected Outcome

The Temperature Converter successfully converts temperature values between Celsius and Fahrenheit, helping users obtain accurate results while demonstrating the practical implementation of basic programming concepts in C.
